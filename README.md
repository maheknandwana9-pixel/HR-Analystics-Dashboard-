# 📊 HR Employee Attrition Analysis using Excel, SQL, Python & Power BI

> An end-to-end HR Analytics project that leverages **Excel, SQL, Python, and Power BI** to analyze employee attrition, uncover business insights, and support data-driven HR decision-making.

---

# 📌 Project Overview

Employee attrition is a critical challenge for organizations as it increases recruitment costs, reduces productivity, and impacts business growth.

This project analyzes the **IBM HR Employee Attrition Dataset** using a complete analytics workflow:

* Microsoft Excel for data preparation
* SQL for business queries
* Python for Exploratory Data Analysis (EDA)
* Power BI for interactive dashboard visualization

The objective is to identify the major factors influencing employee attrition and provide actionable recommendations to improve employee retention.

---

# 🎯 Objectives

* Explore the HR dataset using Microsoft Excel.
* Convert Excel dataset into CSV using Python.
* Perform HR business analysis using SQL.
* Conduct Exploratory Data Analysis (EDA).
* Visualize employee trends using Python.
* Build an interactive Power BI dashboard.
* Generate business insights and recommendations.

# 💼 Key Business Insights

* Employees working overtime have a higher attrition rate.
* Employees with lower job satisfaction are more likely to leave.
* Lower monthly income is associated with higher employee turnover.
* Research & Development records the highest employee attrition.
* Employees with poor work-life balance have a greater chance of leaving.

* Improve employee engagement initiatives.
* Review salary structure regularly.
